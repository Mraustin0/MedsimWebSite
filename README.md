# MedSim

MedSim is a web-based simulator that trains medical students in patient history-taking using the OLDCARTS framework (Onset, Location, Duration, Character, Aggravating, Relieving, Timing, Severity). Students conduct a simulated interview with an AI patient in Thai, then receive detailed feedback scored across each OLDCARTS dimension. Instructors can create custom patient scenarios, monitor student progress, and review session performance from a dedicated dashboard.

Live: [medsim-five.vercel.app](https://medsim-five.vercel.app)
